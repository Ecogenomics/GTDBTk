markers_summary.tsv¶

A summary of unique, duplicated, and missing markers within the 120 bacterial marker set, or the 53 archaeal marker set for each submitted genome.

For each genome:

Unique genes are markers which had hits to only a single gene.

Multiple genes are markers which had hits to multiple genes, where sequences differed.

Multiple unique genes are markers which hits to multiple genes, however, the sequence was identical. These are considered as unique for downstream analysis.

Missing genes are markers which did not have any hits.

  • identify

  • classify_wf
